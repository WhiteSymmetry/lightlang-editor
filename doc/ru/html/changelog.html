0.7.5 beta2
  1. Редактор словарей теперь имеет собственный логотип, отличный от XSL
  2. Сделана IFA (Integratable Friend Application)
  3. Программа показывает прогресс загрузки словаря
  4. Добавлены опции старта программы из консоли:
    • lileditor [путь к словарю] - открывает словарь по указанному пути
    • lileditor [--new] - запускает программу, создав новый словарь
    • lileditor [--clear-cache] [путь к словарю] - чистит кэш заданного словаря, если он загружен
    • lileditor [--remove-block] - удаляет блок-файл, оставшийся в случае некорректного завершения программы
    • lileditor [--help] - просмотр помощи
  5. Изменена логика работы с блок-файлом
  6. Добавлена строка статуса
  7. Сделано хорошее Html руководство
  8. Изменена организация исходников и deb пакета
  9. Иконки и другие изображения, теперь не прикроплены к испоняемому файлу ( не исользуется qrc )
  10. Интерфейс программы переведен на русский язык
  11. Исправлены некоторые мелкие баги (например, после нажатия кнопки "Редактировать" добавлялся лишний символ перевода каретки по бокам блоков)
  12. Теперь можно добавлять словари в базу данных sl одним нажатием кнопки - программа сделает все остальные действия за вас
  13. Теперь нахождение записи во время ввода - опция
