Interaction LightLang with LightLang Editor

How to add edited dictionary to sl database:
If you loaded some dictionary and open it, you can choose point "Add dictionary to XSL". If XSL uses dictionary with the same name, as your, you'll be offered to choose 2 variants: replace and change name ( add int start prefix "New"). Also, you can press "Cancel".
After it, you'll have to enter to "XSL -> manager of dictionaries" and add loaded dictionary to the list of used.

The program LightLang Editor and LightLang is bounded by IFA. If you enter some word in editor, and choose poing "Start Sl", in console will be shown the translation of this word, if editor was started from console. If you choose "Start XSL", graphical interface for sl will be started.
All points will be unactive, if you don't install these programs.
