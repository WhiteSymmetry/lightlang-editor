How to use this program

Consider two situations: 1 - if you have decided to create your dictionary, 2 if you have decided to edit already created dictionary.
1. Move of actions:
  1. Choose "File -> New" ( or Ctrl+N )
  2. Now you can add words: enter to main line a word, which you want to add, then enter translation and format it, how you want and press on the button "Add SL record" ( Ctrl+A )
  3. You can leave from the program and you records will store in database with name "untitled", but if you decide create one more dictionary, you'll have to save old under some name.
  4. To save dictionary, choose "File -> Save" ( Ctrl+S )
  5. After it, you can replace or add created dictionary to database of dictionaries of LightLang - instruction

2. Move of actions:
  1. Choose "File -> Open" ( Ctrl+O ) and open a dictionary. If loaded dictionary is rather big, you'll have to wait some minutes. But you can check on progress, becouse of special progress bar.
  2. After it, as dictionary was loaded, you can edit and create new SL records. How to do it is discussing in point 2 of first situation. Editing some record is rather easy too: enter word in the main line or choose it from list of auto-search, edit translation and press on button "edit" ( Ctrl+E ).
  3. After editing all records will store in special database, but not in a file, that's why, to save dictionary in file, you should use point 4 from second situation.
  4. After saving you can load it to the service LightLang - instruction.

Other functions of the program

Clean cache - it means, remove all files and database, which depend on some loaded dictionary. If you want to edit in dictionary, but you have pressed on this point, you'll have to wait again, while the dictionary will be loaded to special database.
Remove SL record - if you enter existed word, you can remove it by using this button.
Zoom In/Zoom out - if you see badly or letters is big for you, you'll be able to use these functions.
Bookmarks - to don't enter some word, which you use very often, you can add it to bookmarks and by one click on this bookmark you move to editing this word.
Auto-search - this function is very useful, if you don't know how some word is written or if you want watch all words, which is started with such substring, which you have entered. And you can else limit search, using filter. If you have too much words, you can sort them up or down.
